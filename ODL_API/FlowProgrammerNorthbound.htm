FlowProgrammerNorthbound

Flow Configuration Northbound API provides capabilities to program flows.

Authentication scheme : HTTP Basic
Authentication realm : opendaylight
Transport : HTTP and HTTPS

HTTPS Authentication is disabled by default.

The following resources are part of this group:

  • /{containerName}
  • /{containerName}/node/{nodeType}/{nodeId}
  • /{containerName}/node/{nodeType}/{nodeId}/staticFlow/{name}

/{containerName}

Mount Point: /controller/nb/v2/flowprogrammer/{containerName}

GET

Returns a list of Flows configured on the given container

Parameters

name description type default
containerName Name of the Container (Eg. 'default') path

Response Body

element: list
media types: application/xml
application/json

List of flows configured on a given container 

Example:

Request URL:
http://localhost:8080/controller/nb/v2/flowprogrammer/default

Response body in XML:
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<list>
   <flowConfig>
      <installInHw>true</installInHw>
      <name>flow1</name>
      <node>
         <id>00:00:00:00:00:00:00:01</id>
         <type>OF</type>
      </node>
      <ingressPort>1</ingressPort>
      <priority>500</priority>
      <etherType>0x800</etherType>
      <nwSrc>9.9.1.1</nwSrc>
      <actions>OUTPUT=2</actions>
   </flowConfig>
</list>

Response body in JSON:
{
"flowConfig": [
{
"installInHw": "true",
"name": "flow1",
"node": {
"type": "OF",
"id": "00:00:00:00:00:00:00:01"
},
"ingressPort": "1",
"priority": "500",
"etherType": "0x800",
"nwSrc":"9.9.1.1",
"actions": [
"OUTPUT=2"
]
}
]
}

Mount Point: /controller/nb/v2/flowprogrammer/{containerName}/node/{nodeType}/{nodeId}/staticFlow/{name}

  • PUT
  • DELETE
  • POST

Returns the flow configuration matching a human-readable name and nodeId on a given Container.

containerName Name of the Container (Eg. 'default') path
