ControllerManagerNorthbound

The class provides Northbound REST APIs to manager the controller. Currently it supports getting controller property(ies), setting a property, and removing a property

The following resources are part of this group:

  • /configuration
  • /{containerName}/properties
  • /{containerName}/properties/{propertyName}
  • /{containerName}/properties/{propertyName}/{propertyValue}

/configuration

Mount Point: /controller/nb/v2/controllermanager/configuration
