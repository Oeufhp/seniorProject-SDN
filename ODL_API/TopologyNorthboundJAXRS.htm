TopologyNorthboundJAXRS

Topology Northbound REST API

Authentication scheme : HTTP Basic
Authentication realm : opendaylight
Transport : HTTP and HTTPS

HTTPS Authentication is disabled by default.

The following resources are part of this group:

  • /{containerName}
  • /{containerName}/userLinks
  • /{containerName}/userLink/{name}

/{containerName}

Mount Point: /controller/nb/v2/topology/{containerName}

GET

Retrieve the Topology

Parameters

name description type default
containerName The container for which we want to retrieve the topology (Eg. 'default') path

Response Body

element: topology
media types: application/xml
application/json

A List of EdgeProps each EdgeProp represent an Edge of the graph with the corresponding properties attached to it.

/{containerName}/userLinks

Mount Point: /controller/nb/v2/topology/{containerName}/userLinks

GET

Retrieve the user configured links

Parameters

name description type default
containerName The container for which we want to retrieve the user links (Eg. 'default') path

Response Body

element: list
media types: application/xml
application/json

A List of user configured links 

Example:

Request URL:
http://localhost:8080/controller/nb/v2/topology/default/userLinks

Response body in XML:

<status>Success</status>
<name>link1</name>

<dstNodeConnector>OF|2@OF|00:00:00:00:00:00:00:51</dstNodeConnector>

</list>
